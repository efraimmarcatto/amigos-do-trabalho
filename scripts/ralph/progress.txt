## Codebase Patterns
- Passthrough polygon in main.gd `_update_passthrough()` collects Rect2 from all visible UI elements, merges them with Geometry2D, then stitches disjoint islands through Vector2.ZERO anchor for Linux stability
- `Geometry2D.merge_polygons()` can return hole polygons (clockwise winding) — always filter with `Geometry2D.is_polygon_clockwise()` and discard clockwise results
- Git config has broken `gpg.format` — use `GIT_CONFIG_GLOBAL=/dev/null` prefix for all git commands
- No `godot` CLI available in this environment — typecheck cannot be run automatically
- Mood bubble is shown in two places in pet.gd: `_show_mood_bubble()` and the jump-fail frustration block (~line 520) — both must be updated when changing bubble content
- Use `_get_mood_texture(mood)` to get the correct texture for a mood — it checks exported custom textures first, then falls back to generated circles
- Godot plugins live in addons/<name>/ with plugin.cfg + plugin.gd; enable in project.godot [editor_plugins] section
- GlobalInput is now at addons/global_input/ — scripts reference it by autoload singleton name "GlobalInput", not file path
