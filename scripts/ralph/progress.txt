## Codebase Patterns
- Passthrough polygon in main.gd `_update_passthrough()` collects Rect2 from all visible UI elements, merges them with Geometry2D, then stitches disjoint islands through Vector2.ZERO anchor for Linux stability
- `Geometry2D.merge_polygons()` can return hole polygons (clockwise winding) — always filter with `Geometry2D.is_polygon_clockwise()` and discard clockwise results
- Git config has broken `gpg.format` — use `GIT_CONFIG_GLOBAL=/dev/null` prefix for all git commands
